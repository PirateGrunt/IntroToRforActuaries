R workshop at the 2014 RPM

This page will host instructions, data and source code for the R workshop at the 2014 RPM seminar. To register, please visit the CAS website.

This site will be updated continually leading up to the day of the workshop. If possible, I'll notify everyone of changes by e-mail.

10:30-10:45 Break
10:45-12:15 Section 2 - Basic analysis
  • Basic visualization

12:15-1:00 Lunch
1:00-2:30 Section 3 - Reserving in R

4:00-? Further topics
  • Intermediate visualization

  • Reproducible research
