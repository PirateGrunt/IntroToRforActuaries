Welcome to the R workshop at the 2014 RPM!.

This page will host instructions, data and source code for the R workshop at the 2014 RPM seminar. To register, please visit the CAS website. INSERT A LINK HERE.

What will you do in this workshop?

What won’t you do? You’ll hear all about how tremendous R is, where it came from, where it’s going and how it has become the tool of choice for tens of thousands of analysts across the world. You’ll learn how to make sure it runs smoothly despite the objections of your company’s IT department. You’ll learn how to exploit the knowledge of other R users. You’ll learn how to import and export information from text files, Excel files, databases and the web. You’ll draw pictures of the probability density function for loss development factors. You’ll use monte carlo simulation. You’ll learn how to use generalized linear modeling to predict the results of football games. You’ll create a color-coded map to show which states produce poor underwriting results. What else? If time permits, you might use cluster analysis. Or, you might build a word cloud to mine text information from your claims data.

Requirements for this workshop:

  • A functioning laptop (PC or Mac)
  • Installation of R and RStudio in advance of the workshop (instructions and support for this will be provided)
  • Patience, enthusiasm and a sense of humor
